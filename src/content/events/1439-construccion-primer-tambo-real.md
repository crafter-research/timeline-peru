---
date: 1439-08-15
title: Construccion del Primer Tambo Real en Limatambo
category: economia
era: inca
---

Se inicia la construccion del tambo de Limatambo, estableciendo el modelo para la red de postas y almacenes que cubrira todo el Qhapaq Nan. Este tambo incluye depositos para alimentos, cuarteles para tropas y aposentos para funcionarios, sirviendo como prototipo para futuras construcciones similares.

Fuente: El Camino Inca - John Hyslop
