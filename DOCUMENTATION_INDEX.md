## 🎯 GUÍAS RÁPIDAS POR TAREA

### Crear nuevo evento
1. Leer [STYLE_GUIDE.md](./STYLE_GUIDE.md) - Sección 1 y 3
2. Usar template de estructura
3. Verificar con [ACCENT_CHECKLIST.md](./ACCENT_CHECKLIST.md)
4. Consultar categorías/eras en STYLE_GUIDE

### Revisar evento existente
1. Abrir [ACCENT_CHECKLIST.md](./ACCENT_CHECKLIST.md)
2. Verificar ortografía palabra por palabra
3. Expandir si tiene menos de 2 oraciones
4. Añadir fuente con autor + año
5. Validar categoría/era

### Investigar patrones de errores
1. Consultar [IMPROVEMENTS_LOG.md](./IMPROVEMENTS_LOG.md)
2. Sección "Patrones de errores corregidos"
3. Ver ejemplos antes/después

### Planificar siguiente round
1. Revisar [ROUND_3_REVIEW_SUMMARY.md](./ROUND_3_REVIEW_SUMMARY.md)
2. Sección "Recomendaciones futuras"
3. Consultar áreas subrepresentadas en IMPROVEMENTS_LOG

## 📋 CHECKLISTS

### Checklist de calidad (por archivo)
- [ ] Título con acentos correctos
- [ ] Descripción de 2-3 oraciones mínimo
- [ ] Contexto histórico incluido
- [ ] Fuente con autor y año
- [ ] Categoría apropiada
- [ ] Era correcta
- [ ] Nombres propios verificados
- [ ] Ortografía completa revisada

### Checklist de acentuación común
- [ ] "Peru" → "Perú"
- [ ] "ejercito" → "ejército"
- [ ] "maxima/politica/economica" → con tilde
- [ ] "iconografia/ceramica" → con tilde
- [ ] Nombres incas: Cápac, Túpac, Pachacútec
- [ ] Verbos pasado: construyó, inició, etc.


### ¿Cuánto debe medir una descripción?
**Mínimo**: 2-3 oraciones (60-80 palabras)
**Óptimo**: 3-4 oraciones (80-120 palabras)

- **cultura**: Arte, religión, arquitectura, literatura
- **politica**: Gobierno, reformas, instituciones

- **conflictos**: Guerras, desastres naturales
